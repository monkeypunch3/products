Show Desktop
==================

    Show Desktop is an application for easily showing the Mac desktop. All you do is click the icon and 
    all the applications are minimized. 
    
    For more information about Show Desktop please visit http://www.velara3.com/

    Currently supported platforms include:

        Mac OS X
    
    
Install Prerequisites
---------------------

    You must have Mac OS X. 

    
Installation
---------------------

    After you have downloaded and unzipped the software double click the
    ShowDesktop.dmg file. This will launch the full installer. 
    Follow the on screen prompts to complete the installation.
